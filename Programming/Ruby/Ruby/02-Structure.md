# Structures

# Объявление переменных

Динамическая типизация, переменные объявляются так:

```ruby
variable = 10
```

# Условия

## if ... elseif ... else ... end

```ruby
a = 0

if a == 100
    puts "Expression is true, but a is now: #{a}"
elsif a == 20
    puts "Okey"
else
    puts "#{a} is not equal to 100"
end
```
