# Запуск, Import/Export

# Запуск

## Запуск REPL

Для запуска интерактивного режима пишем:

```bash
irb
```

## Запуска скрипта

Запуск скрипта в любом месте:

```bash
ruby path_to_file.rb
```

# Import/Export

## Проверка метода запуска

Файл не импортирован, а запущен, как исполняемый:

```ruby
if __FILE__ == $0
    ...
end
```
